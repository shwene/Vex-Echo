# Vex-Echo
<p>
Welcome to the Memory Flag Challenge! Test your memory and knowledge of national flags in this fun and engaging game. The goal is to click on flags among a group of displayed flags, whilst making sure not to choose the same flag in consecutive rounds. Each correct click earns you points, but be careful ‚Äì repeating a flag selection will reset your point streak!
